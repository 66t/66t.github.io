死灵 Necro （暗能量）

信仰着超越生死界限的致命能量，没有善恶观念的卑劣歹徒，肆意亵渎生命的价值来证明其他存在的荒谬。
- 邪秽 => Unclean
- 疫病 => Plague
- 积怨 => Curse
- 凭依 => Haunt
- 献祭 => Vanish
恐惧之源

防护 Abjuration （生命能量）

通过构建并守护公平可靠的规则，维护平衡和秩序。专注抵御外界的混乱和异动，确保一切归于平和与安宁。
- 力场 => Field
- 剥离 => Detach
- 瓦解 => Crumble
- 干扰 => Interfere
- 抵拒 => Resist
基本法则

  • 聚合 => Aggregation
  • 扩散 => Diffusion
  • 环绕 => Surround
  • 脉冲 => Pulse
  • 传导 => Conduction
    经典范式

非法 Illegal （算力）

  • 计算 => Calc
  • 定位 => Locate
  • 伪装 => Dissimilate
  • 渗透 => Penetrate
  • 革新 => Disruptor
    革新范式

变化 Changes （物之本质）

洞察万物本质，通过分解与重组，物质、意识与存在在恒变中维持内稳态。 变化即秩序，嬗变即恒常。通过实证寻求全知全能的技术神话。
- 转变 => Transform
- 异化 => Mutagenic
- 炼成 => Bonding
- 相变 => Transition
- 变身 => Shapeshift
内稳态

预言 Prophecy （运势）

虚无归一，时光止息，超越时空与尺度。虔诚者与宇宙共鸣触碰命运真相，秩序与混乱的界限逐渐消融，赋予他们对隐匿的命运之掌控。
- 观察 => Observe
- 揭示 => Reveal
- 因果 => Karma
- 先兆 => Omen
- 言灵 => Utter
大设计宇宙
因果关系

  • 心灵 => Mind
  • 操控 => Control
  • 同步 => Synchro
  • 暗示 => Suggest
  • 幻象 => Phantasm
    信息过载 - 偶像
    财富 - 在赌桌上获得公平竞争

咒法 Summon （符号）

掌握基本语言和符号，感知维度间的能量流动。用咒语引导这些能量，操控异界力量为己所用。
- 咒令 => Relocate
- 具现 => Recall
- 创造 => Creation
- 祈求 => Invoked
- 传送 => Teleport
